## **About the Excel Dataset**

This Excel file contains global education statistics compiled from multiple countries and regions.
The dataset includes demographic and development indicators that help highlight inequalities in education access and outcomes across the world.

### **Key Fields in the Dataset**

The dataset covers:

* **Country Information:**

  * ISO3 codes
  * Country names
  * Region and Sub-region
  * Development classification (Least Developed, Less Developed, More Developed)

* **Education Access Indicators:**

  * Total enrolment percentage
  * Female and Male enrolment
  * Rural vs Urban access
  * Enrollment across income groups (Poorest, Second, Middle, Fourth, Richest)

* **Time Period:**

  * Year the data was recorded
  * Education level (e.g., Primary)

### **Purpose of the Dataset**

This dataset was used to:

* Analyze global and regional education inequality
* Compare enrolment gaps between genders, locations, and income levels
